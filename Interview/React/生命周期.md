# 组件的生命周期
React 为组件的不同生命阶段，提供了近十个钩子方法。

- componentWillMount()：组件加载前调用
- componentDidMount()：组件加载后调用
- componentWillUpdate(): 组件更新前调用
- componentDidUpdate(): 组件更新后调用
- componentWillUnmount()：组件卸载前调用
- componentWillReceiveProps()：组件接受新的参数时调用
