# 内存泄漏
## 原因
1. 以外的全局变量
```
function fn() {
a = new Array(10000000)
//window.a = a = new Array(10000000)
console.log(a)
}
fn()
```
可以用 use strict 避免

// 1. 内存溢出
var obj = {}
for (var i = 0; i < 10000; i++) {
obj[i] = new Array(10000000)
console.log('-----')
}

2. 被遗忘的计时器或回调函数
var intervalId = setInterval(function () { //启动循环定时器后不清理
console.log('----')
}, 1000)

// clearInterval(intervalId)

3. 闭包
闭包使得a共享作用域，a得不到释放
function fn1() {
var a = 4
function fn2() {
console.log(++a)
}
// a = null 可以解决
return fn2
}
var f = fn1()
f()
